Formação SQL Database Specialist - DIO 🎓

Este repositório contém os projetos e desafios práticos desenvolvidos durante a minha formação na Digital Innovation One (DIO). O objetivo é consolidar conhecimentos em modelagem de dados, criação de bancos de dados relacionais, otimização de queries e automação com Stored Procedures.

🛠️ Tecnologias e Ferramentas
SGBD: MySQL

Modelagem: MySQL Workbench

Linguagens: SQL (DDL, DML, DQL, DTL)

📂 Projetos no Repositório
1. Sistema de E-commerce Completo
Este projeto envolveu a criação de um ecossistema de e-commerce desde o zero, tratando a complexidade de múltiplos fornecedores, estoques e tipos de clientes (PF/PJ).

Destaques: * Modelagem EER (Enhanced Entity-Relationship).

Implementação de restrições (Constraints) e chaves primárias/estrangeiras.

Queries complexas com INNER JOIN, GROUP BY e filtragem com HAVING.

Arquivos: Script BD(Prompt Banco de Dados).sql, Script BD(Prompt ADD Users).sql.

2. Gestão de Oficina Mecânica
Criação de um banco de dados para controle de serviços, peças, fornecedores e histórico de garantia em uma oficina.

Destaques:

Uso de tipos ENUM para categorias de peças.

Lógica de cálculo de lucro bruto e controle de estoque.

Consultas de acompanhamento de prazos de garantia.

Arquivo: DB Oficina.sql.

3. Otimização e Automação (Procedures & Índices)
Implementação de lógica de programação dentro do banco de dados para facilitar a manipulação de dados e melhorar a performance.

Destaques:

Stored Procedures: Criação de uma procedure (sp_manipula_produto) que centraliza operações de CRUD (Insert, Update, Delete) usando blocos CASE.

Índices (B-Tree): Criação de índices otimizados para acelerar buscas em tabelas de empregados e departamentos.

Arquivos: parte1_ecommerce.sql, parte2_ecommerce.sql.

🚀 Competências Demonstradas
Modelagem de Dados: Transformação de requisitos de negócio em diagramas lógicos e físicos.

Consultas Avançadas: Extração de relatórios inteligentes para tomada de decisão.

Manutenibilidade: Uso de scripts SQL organizados e comentados.

Performance: Implementação de índices para ganho de desempenho em grandes volumes de dados.
